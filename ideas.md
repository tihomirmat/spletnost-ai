## Idea 1: "Cosmic Command Center"

**Design Movement:** Dark Tech Brutalism meets Space Opera

**Core Principles:**
1. Dark, immersive backgrounds with cosmic depth (deep navy/black with purple nebula gradients)
2. Orange as the "ignition" color — used sparingly but powerfully for CTAs and highlights
3. Asymmetric layouts with bold diagonal cuts and overlapping elements
4. Data-driven visual language: grids, circuits, glowing lines

**Color Philosophy:**
Dark space background (#0A0614) as the canvas. Orange (#F84C00) as the "fire" of automation — energy, action, urgency. Purple (#4A00EB) as the "intelligence" — depth, mystery, AI. White text for clarity against darkness.

**Layout Paradigm:**
Full-width sections with diagonal clip-paths between them. Hero section is asymmetric — text on left, animated visual on right. Cards float with subtle glow effects. Sticky navigation with blur backdrop.

**Signature Elements:**
- Glowing orange/purple circuit-line decorations
- Star field particle backgrounds
- Rocket/launch iconography for CTAs

**Interaction Philosophy:**
Hover states reveal hidden details. Cards "launch" on hover (slight upward translate). Numbers count up on scroll. Sections fade in from below.

**Animation:**
Entrance animations for each section (fade + slide). Continuous subtle star particle movement in hero. Hover glow pulses on cards.

**Typography System:**
- Display: Montserrat 800 (ultra-bold, uppercase for section titles)
- Body: Source Sans Pro 400/600
- Accent: Montserrat 600 italic for taglines

## Idea 2: "Industrial Precision"

**Design Movement:** Swiss Grid Modernism + Industrial Tech

**Core Principles:**
1. Rigid grid structure with deliberate rule-breaking moments
2. Orange as a structural element, not just decoration
3. Heavy typographic hierarchy — size contrast is the primary design tool
4. Minimal decoration, maximum information density

**Color Philosophy:**
White (#FFFFFF) background with orange (#F84C00) as structural dividers and highlights. Dark charcoal (#1A1A2E) for text and dark sections. Purple used only as a subtle accent.

**Layout Paradigm:**
Newspaper-style grid with varying column widths. Large numbers (01, 02, 03) as section markers. Horizontal rules and vertical lines create structure.

**Signature Elements:**
- Large bold section numbers
- Orange horizontal rule dividers
- Monospace code-style text for technical details

**Interaction Philosophy:**
Precise, intentional. Hover states are sharp (no blur). Transitions are quick (150ms). Everything feels engineered.

**Animation:**
Slide-in from left for text. Counters for statistics. Minimal — every animation has a purpose.

**Typography System:**
- Display: Montserrat 900 (massive, structural)
- Body: Source Sans Pro 400
- Code: JetBrains Mono for technical elements

## Idea 3: "Nebula Flow" ← SELECTED

**Design Movement:** Cosmic Luxury + Fluid Tech

**Core Principles:**
1. Deep space dark background with flowing gradient nebulas (orange-to-purple)
2. Glass morphism cards floating in space — frosted glass with subtle borders
3. Fluid, organic shapes contrasting with sharp technical typography
4. Generous whitespace within dark sections creates premium feel

**Color Philosophy:**
Background: Deep dark navy-black (#080B1A). Primary gradient: Orange #F84C00 → Purple #4A00EB (the "nebula gradient"). Cards: rgba(255,255,255,0.05) with 1px rgba(255,255,255,0.1) border. Text: Pure white for headings, rgba(255,255,255,0.7) for body.

**Layout Paradigm:**
Full-bleed dark sections with organic blob/nebula backgrounds. Cards in a staggered grid (not uniform). Hero section has large text left-aligned with a glowing visual element right. Sections separated by gradient fade rather than hard lines.

**Signature Elements:**
- Glowing gradient text (orange-to-purple) for key phrases
- Glassmorphism cards with subtle inner glow
- Floating particle/star dots in backgrounds

**Interaction Philosophy:**
Smooth and fluid. Cards lift and glow on hover. Gradient shifts slightly on mouse movement. Everything feels alive but not chaotic.

**Animation:**
Framer Motion entrance animations (staggered fade-up). Gradient orbs slowly pulse in background. Hover: card lifts 4px + glow intensifies. Scroll-triggered counter animations.

**Typography System:**
- Display: Montserrat 800/900 — large, commanding
- Body: Source Sans Pro 400/600 — clean, readable
- Gradient text: key phrases in orange-to-purple gradient
- Size scale: 72px hero → 48px section → 32px subsection → 18px body

## Selected Design: "Nebula Flow"

Cosmic Luxury + Fluid Tech approach — dark space backgrounds with flowing orange-to-purple nebula gradients, glassmorphism cards, and Montserrat/Source Sans Pro typography. This perfectly aligns with Spletnost's existing space/rocket brand identity while elevating it to a premium AI product feel.
